🎤 Python Voice Assistant


Project Description

      The Python Voice Assistant is a software application that can understand human voice commands      and perform different tasks automatically. The assistant listens to the user's voice through a    microphone, converts speech into text, processes the command, and responds using voice output.
             This project demonstrates how voice recognition and automation can be implemented using Python
Objective
     The main objective of this project is to develop a voice-controlled assistant that can understand voice commands and perform tasks such as opening websites, playing music, taking notes, and checking weather information.
Technologies Used
   
     - Python
    - SpeechRecognition
    - PyAudio
    - pyttsx3
    - pywhatkit
    - requests
    - wikipedia
    - pyautogui
 Features
    - Open websites like YouTube, Google, Instagram and GitHub
    - Play songs on YouTube using voice commands
    - Tell the current time
    - Save notes using voice
    - Take screenshots
    - Send WhatsApp messages
    - Get weather information using API
- Voice-based interaction with the system
How the System Works

   1. The user gives a voice command using the microphone.
   2. The SpeechRecognition library converts the voice into text.
   3. The program processes the command using Python conditions.
   4. Based on the command, the assistant performs the requested task.
   5. The assistant responds using Text-to-Speech technology.
